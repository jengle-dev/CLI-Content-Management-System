# CLI-Content-Management-System
A command-line application from scratch to manage a company's employee database, using Node.js, Inquirer, and MySQL.

## Installation
* npm init -y
* npm i
* npm i inquirer@8.2.4
* npm install console-table-printer --save
* npm install mysql2
